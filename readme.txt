Growing collection of marketing widgets for the Divi 3.1 visual builder!

== Description ==

This plugin extends the Divi Theme with new, custom modules for the Visual Builder. Note, that you need to have at least Divi with version 3.1.0 in order to use this plugin!

== Installation ==

Install the plugin from the WordPress admin page "Plugins > Install"

or

1. Upload `marketing-with-divi` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
